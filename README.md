# @musakui/create-vite

Create an app powered by [Vite](https://github.com/vitejs/vite).

```bash
$ npm init @musakui/vite <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

## Templates

Available templates:

- `vue` (default)
- `three` (Three.js)
- `react`
- `react-ts`
- `preact`
- `reason-react`

To scaffold with specific template:

```bash
$ npm init @musakui/vite my-three-project --template three
```
